HomeDecor — a responsive landing page for showcasing products and solutions in the field of decor and interior design. The project is focused on a modern UI/UX approach with an emphasis on visual content presentation, smooth user interaction with the interface, and proper display across different devices.

## 🔗 Live Demo

[HomeDecor](https://taras-kozii.github.io/HomeDecor/)

## 🚀 Core Technologies

- HTML5 — semantic page structure
- CSS3 — styling, responsive layout, and UI animations
- JavaScript (Vanilla JS) — interactivity and interface behavior
- Responsive Web Design (RWD) — adaptation for mobile/tablet/desktop
- Flexbox & CSS Grid — building a modern layout system
- CSS Transitions & Animations — smooth hover effects and transitions
- Git & GitHub Pages — version control and project deployment

## 📱 Responsive Design

The landing page is fully responsive and optimized for:
- smartphones
- tablets
- laptops
- desktop devices

- minimization of unnecessary DOM elements;
